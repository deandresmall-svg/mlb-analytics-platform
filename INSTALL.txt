STATCAST NULL-INTEGER FIX

From the project root:

  unzip -o mlb-phase2-statcast-na-fix.zip
  python -m py_compile src/mlb_analytics/data/statcast.py
  pkill -f streamlit
  streamlit run app/Home.py --server.port 8502

Then retry one completed date on Data Sources with Days per download chunk = 1.

This fix treats missing Statcast zone/launch measurements as false indicator flags instead of attempting to convert pandas NA values directly to integers.

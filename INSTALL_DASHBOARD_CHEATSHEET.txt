MLB DASHBOARD + AUTO-UPDATING CHEAT SHEET
==========================================

This update does not replace the database, trained model files, Statcast tables,
or Odds API configuration.

Install from the project root:

  cd /workspaces/mlb-moneyline-model-
  unzip -o mlb-dashboard-cheatsheet-update.zip

Restart Streamlit:

  source .venv/bin/activate
  pkill -f streamlit || true
  python -m streamlit run app/Home.py --server.port 8502 --server.address 0.0.0.0

After installation:

1. Open Calibration.
2. Click "Run holdout reliability evaluation" once.
3. The evaluation writes data/reliability_snapshot.json.
4. Open the new Cheat Sheet page, Home tabs, or Best Bets.

The cheat sheet updates after every completed reliability evaluation. If models
are retrained afterward, the dashboard marks the saved cheat sheet as stale until
Calibration is run again.
